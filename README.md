# ☕ The Golden Cup Café — Website

> **Future Interns | Full Stack Web Development Internship — Task 3 (2026)**

A fully responsive, professional website built for **The Golden Cup**, a fictional warm neighbourhood café based in Bandra West, Mumbai.

🔗 **Live Site:** [https://t29bhoomi.github.io/FUTURE_FS_03/](https://t29bhoomi.github.io/FUTURE_FS_03/)

---

## 📸 Preview

| Section | Description |
|---|---|
| 🏠 Hero | Animated coffee cup, tagline, CTA buttons |
| 📊 Stats Strip | Key highlights — customers, menu items, years |
| 🫖 About | Story, values, and café features |
| 🍽️ Menu | 12 items across Coffee, Food & Cold Drinks with live filters |
| ⭐ Testimonials | Customer reviews with star ratings |
| 🖼️ Gallery | Asymmetric 5-panel photo grid |
| 📍 Contact | Address, hours, WhatsApp button, contact form & map |

---

## ✨ Features

- ✅ Fully responsive — works on mobile, tablet & desktop
- ✅ Interactive menu with category filter tabs (All / Coffee / Food / Cold Drinks)
- ✅ Smooth scroll animations on page load and scroll
- ✅ WhatsApp direct chat button
- ✅ Working contact form with toast notification
- ✅ Fixed navbar with scroll shadow effect & mobile hamburger menu
- ✅ Google Maps placeholder (ready to embed)
- ✅ Zero dependencies — pure HTML, CSS & JavaScript
